# Banco de Dados Relacional em Python com sqlite3

Este projeto demonstra a implementação de um banco de dados relacional em Python, explorando todos os principais tópicos do SQL com `sqlite3`. O repositório está organizado em pastas temáticas que abordam desde conceitos básicos do SQL, até funções avançadas, triggers, views e buscas em texto.

---

## Estrutura do Projeto

Cada pasta representa um conceito chave do SQL ou da modelagem de dados. Todos os arquivos utilizam a biblioteca **sqlite3** para manipular bancos de dados locais. Abaixo, você pode ver o que encontrará em cada diretório:

| Pasta                        | Conteúdo Principal                                                                   |
|------------------------------|--------------------------------------------------------------------------------------|
| **Basico**                   | Introdução ao SQL: criação, consulta, alteração e remoção de tabelas e dados         |
| **Criptografia**             | Técnicas e funções para proteger dados salvos no banco                               |
| **DDL**                      | Data Definition Language: comandos para criar, alterar e apagar estruturas           |
| **DML**                      | Data Manipulation Language: comandos para inserir, atualizar e remover registros     |
| **Funcao_classificação**     | Funções de classificação e ordenação de dados (ORDER BY, ranking etc.)               |
| **Funcao_informacao**        | Consultas informativas, extração de metadados, estatísticas                         |
| **Funcao_logica**            | Funções lógicas e condicionais (CASE, IFNULL, operadores lógicos)                   |
| **Funcao_matematica**        | Funções matemáticas nativas do SQLite                                                |
| **Funcoes_agregadas**        | Funções agregadoras: SUM, AVG, COUNT, MIN, MAX                                      |
| **Funcoes_caracteres**       | Manipulação de strings: UPPER, LOWER, SUBSTR, etc.                                  |
| **Funcoes_conversao**        | Conversão de tipos de dados nas queries                                              |
| **Funcoes_dataHora**         | Manipulação de datas e horas (DATE, TIME, DATETIME)                                 |
| **Functions**                | Funções Python personalizadas usadas junto ao banco SQLite                           |
| **Modelagem**                | Exemplos de modelagem relacional: entidades, relacionamentos, diagramas, normalização|
| **Procedures**               | Simulações de rotinas automatizadas, modelo de procedures usando Python              |
| **TextSearch**               | Pesquisa textual avançada com FTS (Full Text Search) do SQLite                       |
| **Tiggers**                  | Gatilhos: automação de ações (TRIGGER)                                              |
| **Union_subquey_join**       | Consultas avançadas: UNION, subqueries, JOIN                                        |
| **Views_sql**                | Criação e uso de views para abstração de consultas                                  |
